WalletRepository

WalletRepository class Null safety

A repository that provides all the necessary to communicate with Dero wallet.

You need to supply your wallet rpc server address when instantiating WalletRepository. And before calling the available RPC methods, you also need to start the client :

final WalletRepository walletRepository = WalletRepository('127.0.0.1:10103');

walletRepository.start();
