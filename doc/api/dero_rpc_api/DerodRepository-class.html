DerodRepository

DerodRepository class Null safety

A repository that provides all the necessary to listen and communicate with derod (Dero daemon).

You need to supply your derod rpc server address when instantiating DerodRepository. And before calling the available RPC methods, you also need to start the client :

final DerodRepository derodRepository = DerodRepository('127.0.0.1:10102');

derodRepository.start();
